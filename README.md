# chesslib
chess game processing lib maven repo.

# usage
```
<repositories>
    <repository>
        <id>chesslib-mvn-repo</id>
        <url>https://raw.github.com/dendiz/chesslib/mvn-repo/</url>
        <snapshots>
            <enabled>true</enabled>
            <updatePolicy>always</updatePolicy>
        </snapshots>
    </repository>
</repositories>
```
